# 前端任务清单（基于缺失功能优先级设计文档）

来源文档：`docs/superpowers/specs/2026-04-21-missing-features-priority-design.md`

状态说明：
- `已完成`：前端代码已经落地，主路径基本可用
- `部分完成`：已有页面、交互或接口接入，但仍有缺口未闭环
- `未完成`：当前前端代码中尚未落地

## P0

- [已完成] Dashboard 时间维度切换
  备注：`src/views/DashboardView.vue` 已支持 `本月 / 近 3 月 / 本年` 切换，并会随切换重新拉取统计数据。

- [部分完成] Dashboard 统计模块统一承接
  备注：已具备收支概览、分类占比、消费趋势、预算执行和跳转到账单页、预算页的入口；去分类页的入口未看到，失败态展示也不完整。

- [部分完成] Dashboard 加载态、空态、失败提示补齐
  备注：已有全屏加载和多个空态；接口失败时主要是控制台报错和局部数据清空，缺少明确的页面级失败提示。

- [部分完成] 统计页下线
  备注：`/statistics` 路由已经移除，底部导航也没有统计页入口；但 `src/views/StatisticsView.vue` 文件仍在，仍需清理或停用。

- [已完成] 账单页分类筛选
  备注：`src/views/BillView.vue` 已有分类下拉筛选，并向 `getFinanceTransactionsList` 传递 `categoryId`。

- [已完成] 账单页关键词搜索
  备注：`src/views/BillView.vue` 已支持按备注或分类关键词搜索，并向列表接口传递 `keyword`。

- [已完成] 账单页加载更多交互
  备注：已有 `pageNum / pageSize / hasMore / loadMore` 逻辑，界面也已落地“加载更多”和“没有更多账单了”结束态。

- [已完成] 账单页整理结果反馈
  备注：已支持搜索无结果空态、无更多数据结束态，以及编辑/删除成功后的列表刷新。

- [已完成] 个人资料页与路由
  备注：`/settings/profile` 路由、设置入口和 `src/views/settings/ProfileView.vue` 页面均已存在。

- [部分完成] 个人资料保存后数据回显闭环
  备注：页面已接入 `getUserInfo` 和 `updateInfo`，保存后会更新本地 store；是否完全以后端保存结果为准，仍建议联调再确认。

- [已完成] 通知设置页与路由
  备注：`/settings/notifications` 路由、设置入口和 `src/views/settings/NotificationSettingsView.vue` 页面均已存在。

- [部分完成] 通知设置保存与回显闭环
  备注：已支持邮箱、催记账邮件、月度摘要邮件、发送时间偏好，并通过 `getUserInfo/updateInfo` 读写；字段是否与后端完全对齐仍需联调确认。

- [已完成] 分类删除前端对接
  备注：`src/views/settings/CategoriesView.vue` 已调用 `deleteCategory`，并优先展示后端返回的业务失败原因。

- [部分完成] 设置总入口仅承担导航职责
  备注：分类、预算、个人资料、通知设置都已作为子页面入口存在；但文档中提到的数据管理相关入口尚未补齐。
