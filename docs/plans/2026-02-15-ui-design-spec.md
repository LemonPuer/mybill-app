## 10. 图表配色（动态方案）

分类颜色采用动态生成，基于 HSL 色环均匀分布：

```typescript
// 黄金角分布算法，确保颜色均匀不重复
const getCategoryColor = (index: number, isDark: boolean): string => {
  const hue = (index * 137.508) % 360
  const saturation = isDark ? '70%' : '65%'
  const lightness = isDark ? '65%' : '55%'
  return `hsl(${hue}, ${saturation}, ${lightness})`
}
```

**特点：**

- 自动适配任意数量分类
- 颜色整体和谐，辨识度高
- 自动适配明暗模式

## 11. 页面结构

### 11.1 路由配置

| #   | 页面             | 路由          | 说明                     |
| --- | ---------------- | ------------- | ------------------------ |
| 1   | 首页（智能记账） | `/`           | 文字输入转账单，智能解析 |
| 2   | 仪表盘           | `/dashboard`  | 数据概览                 |
| 3   | 账单详情         | `/bills`      | 账单列表/详情            |
| 4   | 统计报表         | `/statistics` | 数据分析                 |
| 5   | 管理设置         | `/settings`   | 分类/预算/主题等         |

### 11.2 首页（智能记账）

**交互流程：**

```
用户输入文字 → 调用后端解析API → 返回解析结果 → 弹窗确认 → 确认后新增账单
```

**界面要素：**

- 大输入框（类似Google/Dify对话式）
- 智能解析结果预览
- 确认弹窗（显示解析的金额、分类、日期等）
- 支持编辑修改后确认

**交互说明：**

- 输入框支持多行文字
- 回车或点击按钮提交
- 解析成功后弹窗展示解析结果
- 用户确认后调用新增账单接口

### 11.3 仪表盘

**界面要素：**

- 本月收入/支出/结余 统计卡片
- 近期账单列表（最近3-5条）
- 消费分类占比饼图
- 预算执行情况
- 快捷分类入口（点击跳转对应筛选）

### 11.4 账单详情页

**功能清单：**

- [x] 账单列表（按日期分组）
- [x] 筛选（日期范围、分类、账户、金额范围）
- [x] 搜索（按备注/金额搜索）
- [x] 编辑账单
- [x] 删除账单
- [x] 下拉刷新 + 上拉加载更多
- [x] 添加账单

**交互说明：**

- 列表支持左滑编辑/删除
- 筛选采用筛选器面板
- 支持日期快捷选择（今天/昨天/本月/上月）

### 11.5 统计报表页

**功能清单：**

- [x] 收支趋势图（按月/按年切换）
- [x] 分类占比饼图
- [x] 分类对比柱状图
- [x] 账户资产分布
- [x] 年度账单对比（支持月/年切换）

**交互说明：**

- 图表支持时间范围切换
- 数据支持导出（如需要可后续扩展）

### 11.6 管理设置页

**功能模块：**

## 14. 状态设计

### 14.1 空状态

**设计规范：**

- 文字居中显示
- 显示文案：`暂无数据`
- 不显示插画，保持简洁

### 14.2 加载状态

**设计规范：**

- 使用 **骨架屏 (Skeleton)** 方案
- 灰色占位区块，模拟内容布局
- 动画: 轻微闪烁效果

```vue
<el-skeleton :rows="3" animated />
```

### 14.3 页面过渡动画

**实现方案：**

```css
/* 淡入 + 微滑动 */
.page-enter-active,
.page-leave-active {
  transition: all 300ms ease-out;
}

.page-enter-from {
  opacity: 0;
  transform: translateY(20px);
}

.page-leave-to {
  opacity: 0;
  transform: translateY(-20px);
}
```

- 进入动画: 从下往上淡入
- 离开动画: 从上往下淡出
- 时长: 300ms
- 缓动: ease-out
