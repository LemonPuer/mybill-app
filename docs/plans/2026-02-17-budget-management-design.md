# 预算管理页面设计

> 日期: 2026-02-17

## 概述

实现设置页面的预算管理功能，用户可以按分类和时间段设置预算。

## 路由结构

- 路径: `/settings/budget`
- 面包屑: 设置 > 预算管理

## 页面设计

### 布局

- 顶部面包屑导航 + 返回按钮 + 添加按钮
- 时间范围筛选
- 预算卡片列表

### 预算卡片

- 分类名称 + 图标
- 预算金额
- 时间范围
- 已使用/预算金额（进度条）
- 操作按钮（编辑、删除）

### 弹窗

- 选择分类 + 输入金额 + 选择时间范围
- 支持快捷选项：当月、当年
- 删除确认

## API

- 获取预算: `getBudgetInfo(startTime, endTime)`
- 保存预算: `saveBudget({ categoryId, amount, startTime, endTime })`
- 删除预算: `deleteBudget(id)`

## 默认行为

- 默认查询未归档数据（endTime >= 当前时间）

## 样式规范

- 与分类管理页面保持一致
