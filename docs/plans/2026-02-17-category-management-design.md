# 分类管理页面设计

> 日期: 2026-02-17

## 概述

实现设置页面的分类管理功能，用户可以在独立页面中管理收支分类。

## 路由结构

- 路径: `/settings/categories`
- 面包屑: 设置 > 分类管理

## 页面设计

### 布局

- 顶部面包屑导航 + 返回按钮
- 分类卡片网格布局（2列）
- 右下角新增分类悬浮按钮

### 分类卡片

- 图标（圆角方形背景，主色调）
- 分类名称
- 操作按钮（编辑、删除）

### 弹窗

- 新增/编辑：图标选择 + 名称输入
- 删除：二次确认

## API

- 获取分类: `getCategory` (已有)
- 保存分类: `saveCategory` (已有)
- 删除分类: 新增 `deleteCategory`

## 样式规范

- 遵循项目现有主题系统 (CSS变量)
- 圆角: 20px (卡片), 12px (图标)
- 阴影: var(--shadow-card)
- 颜色: 使用 var(--color-primary) 系列
- 响应式: 手机单列 / PC双列
